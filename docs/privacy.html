오늘의모험 개인정보처리방침

시행일: 2026년 7월 17일
오늘의모험은 어떠한 개인정보도 수집하지 않습니다. 모든 데이터는 사용자의 기기 안에만 저장되며, 외부 서버로 전송되지 않습니다.

1. 수집하는 정보

오늘의모험(이하 "앱")은 서버가 없는 오프라인 앱입니다. 앱은 다음을 수집하지 않습니다.

  • 이름, 이메일, 전화번호 등 개인 식별 정보
  • 위치 정보
  • 기기 식별자, 광고 식별자
  • 사용 기록, 분석 데이터

2. 기기 안에 저장되는 데이터

앱 사용 중 입력하는 데이터(아이 이름 또는 별명, 퀘스트 기록, 경험치, 보호자 잠금 번호)는 모두 사용자 기기의 로컬 저장소에만 저장됩니다. 이 데이터는 개발자를 포함한 그 누구에게도 전송되지 않으며, 앱을 삭제하면 함께 삭제됩니다.

보호자 잠금 번호(PIN)는 평문이 아닌 해시 형태로 저장됩니다.

3. 데이터 백업

보호자 화면의 백업 기능은 데이터를 JSON 파일로 만들어 사용자가 직접 선택한 위치에 저장하도록 합니다. 백업 파일의 보관과 관리는 전적으로 사용자에게 있으며, 개발자는 백업 파일에 접근할 수 없습니다.

4. 광고 및 제3자 서비스

  • 앱에는 광고가 없습니다.
  • 분석 도구, 추적 SDK 등 제3자 서비스를 사용하지 않습니다.
  • 인앱 결제가 없습니다.

5. 아동의 개인정보

이 앱은 어린이가 보호자와 함께 사용하도록 만들어졌습니다. 앱은 어린이를 포함한 어떤 사용자의 개인정보도 수집·전송하지 않으므로, 아동 개인정보가 외부에 노출될 위험이 없습니다.
